Privado — Bilingual Translator

### Short Description (≤132 chars)
```
Read any webpage in two languages at once. Clean bilingual translation with YouTube subtitles. Private, no bloat.
```

### Long Description
```
Privado gives you clean, side-by-side bilingual translation on any webpage — original text above, translation below, no clutter.

CORE FEATURES

• Bilingual paragraph view — Every paragraph shows the original and translation together. Read at your own pace.
• YouTube bilingual subtitles — Watch any video with dual-language captions synced in real time.
• Three display modes — Switch between Bilingual, Translation Only, or Original Only with one click.
• Per-site control — Disable translation on sites you don't need it.
• Multiple target languages — Chinese, English, Japanese, Korean, French, German, Spanish, Russian, Arabic, and more.

TRANSLATION MODES

• Local (Private) — Uses Chrome's built-in AI model (Gemini Nano) directly on your device. Completely offline, no data leaves your computer. Available on Chrome 138+ on supported hardware.
• Machine (Free) — Automatic cloud fallback when local AI isn't available. No setup required.
• API Key — Connect your own DeepL, OpenAI, GPT-4, Google Gemini, or any OpenAI-compatible endpoint for higher quality. Your key is stored locally and goes only to your chosen provider.

WHAT WE DON'T DO

✗ No ads injected
✗ No tracking or analytics
✗ No subscription or paywall
✗ No translation of navigation bars, footers, or ad units
✗ No data sent to our servers — ever

SMART FILTERING

Privado only translates meaningful content. It skips navigation menus, footers, ads, code blocks, and short UI labels — so pages stay clean and readable.

OPEN SOURCE

Privado is fully open source (MIT License). Review the code, report issues, or contribute at github.com/wanqianwin-jpg/Privado-Bilingual-Translator
```

## 简体中文

### 名称 (≤45 chars)
```
Privado · 双语翻译
```

### 简短描述 (≤132 chars)
```
任意网页一键双语对照，原文译文同时显示。支持 YouTube 双语字幕，无广告，不收集数据。
```

### 详细描述
```
Privado 为任意网页提供干净的双语对照翻译——原文在上，译文在下，排版整洁，无干扰。

核心功能

• 双语段落对照 — 每段原文下方显示译文，随时切换阅读节奏
• YouTube 双语字幕 — 实时同步原文与译文字幕，支持所有有字幕的视频
• 三种显示模式 — 双语对照 / 仅译文 / 仅原文，一键切换
• 网站级开关 — 对特定网站关闭翻译，设置永久生效
• 多目标语言 — 中文（简繁）、英语、日语、韩语、法语、德语、西班牙语、俄语、阿拉伯语等

翻译模式

• 本地翻译（推荐） — 使用 Chrome 内置 AI 模型（Gemini Nano）在本地设备完成翻译，完全离线，零数据外传。需 Chrome 138+ 及支持的硬件。
• 机翻（免费兜底） — 本地模型不可用时自动切换云端翻译，无需任何配置。
• API Key 模式 — 接入自己的 DeepL、OpenAI、Google Gemini 或任意 OpenAI 兼容接口，追求更高翻译质量。Key 仅存于本地，直接请求你选择的服务商。

我们不做的事

✗ 不注入广告
✗ 不追踪用户行为，不上报统计
✗ 无订阅，无付费墙
✗ 不翻译导航栏、页脚和广告区域
✗ 数据不经过我们的服务器

智能过滤

Privado 只翻译有意义的正文段落，自动跳过导航菜单、页脚、广告、代码块和短 UI 文字，页面保持整洁可读。

开源项目

Privado 完全开源（MIT 协议）。查看代码、提交问题或参与贡献：
github.com/wanqianwin-jpg/Privado-Bilingual-Translator
```
